Селектори!

  1. *
    • (універсальний селектор) - Звернення до всіх єлементів.
  2. h1
    • (селктор елементу) - використовуються для визначення стилів елементів для всіх сторінок веб-сайту.
  3. .class
    • (селектор класу) - може звертатися до декількох елементів маючи однаковий клас.
  4. #id
    • (селектор ідентифікатору) - використовуються для привласнення стилю одному конкретному елементу.
  5. #id,.class,h1
    • (груповий селектор) - слугує для корочення кількості коду в CSS файлі. З допомогою групового селектору можем задати якесь загальне значення для декількох елементів.
  6. ul span
    • (селектор нащадків) - можна варіативно застосувати CSS стилі до потрібних елементів, пославшись на їх батьківський елемент
  7. ol>li
    • (дочірній селектор)- вибирає лише елементи, на які посилається перший селектор.
  8. h1+p
    • (cусідній селектор) - коли вони йдуть безпосередньо один за одним в коді документа
  9. h1~p
    • (cпоріднені селектори) - застосовуються до всіх довколишніх елементів. Наприклад, для селектора h1~p стиль буде застосовуватися до всіх елементів p
  10. [href*="https"] або [href$="https"] або [href^="https"]
    • (вибірковий селектор) - задається збіг по значенню атрибута, містить..., починається на..., закінчується на...
